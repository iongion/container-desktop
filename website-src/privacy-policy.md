# Privacy policy

<p class="lead">Container Desktop is local-first: it performs no telemetry and collects, stores or transmits no data of its own.</p>

- This project does not perform any telemetry.
- This project does not collect, store or process any data, personal or 3rd party.
- Accessing 3rd-party registries such as docker.io or quay.io have their own behavior in relation to privacy.
- This website uses GitHub Pages, which performs its own analytics — see the [GitHub general privacy statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).
